# Nic-Unlocker

NicUnlocker is a direct registry level NIC optimization utility that automatically detects your network adapter and applies the best Configuration unlocking the full potential of your hardware by tuning driver settings, interrupt behavior, and network parameters at the deepest OS level.

It supports Intel, Realtek, Killer, Broadcom, and Marvell adapters, automatically updates the NIC driver to the latest version, and applies all changes instantly without requiring a reboot. Interrupt moderation is removed, checksum and send offloads are configured, transmit and receive buffers are set to their optimal values, and RSS is configured. The optimal MTU is discovered and applied automatically. Flow control, QoS, energy efficient ethernet, wake-on-LAN, sleep-on-disconnect, and all power-saving features are fully disabled to ensure maximum responsiveness at all times.
